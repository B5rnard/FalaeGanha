# FalaeGanha - Image Configuration Guide

This document lists all sentences in the game and their corresponding image URLs. Use this guide to systematically find and update images for each scene.

## How to Update Images

1. Visit [Unsplash](https://unsplash.com) or another image source
2. Search for images matching the descriptions below
3. Copy the image URL
4. Open `js/game.js` and replace the placeholder URL with the real image URL
5. Test the game to ensure images load correctly

## Image Requirements

- **Recommended size**: 800x600px or similar landscape ratio
- **Format**: JPG or PNG
- **Style**: Clear, bright, child-friendly images
- **Quality**: High resolution for clarity on all devices

## Testing Your Images

After updating URLs in `js/game.js`:

1. Open `index.html` in your browser
2. Play through all 5 scenes to verify images load correctly
3. Check that images display well on different screen sizes
4. Ensure images are appropriate and clear for your son

## Tips for Finding Good Images on Unsplash

1. Use the free license images (Unsplash License)
2. Look for images with good composition and lighting
3. Choose images that clearly show the action or location
4. Prefer bright, colorful images that appeal to children
5. Right-click on the image and select "Copy Image Address" to get the URL
6. Use the "Download" button to get high-resolution versions if needed

## Notes

- All placeholder URLs currently point to temporary images
- The game will show a fallback image if any URL fails to load
- Loading indicators appear while images are being fetched
- Images are responsive and will adjust to different screen sizes automatically
