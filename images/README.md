# Images Folder

This folder contains all the images used in the FalaeGanha game.

## Required Images

Place the following 5 image files in this folder:

1. **boy-eating-pizza.png** - Young boy eating pizza
2. **girl-drinking-juice.png** - Young girl drinking juice
3. **dog-running-park.png** - Dog running in a park
4. **car-in-garage.png** - Car parked in a garage
5. **book-on-table.png** - Book on top of a table

## Image Specifications

- **Format**: PNG or JPG (PNG recommended for best quality)
- **Dimensions**: 800x600px or similar landscape ratio (16:9 or 4:3)
- **File size**: Keep under 500KB per image for fast loading
- **Style**: Clear, bright, child-friendly images

## How to Add Images

### Option 1: Generate with ImageFX (Recommended)
1. Visit [Google ImageFX](https://aitestkitchen.withgoogle.com/tools/image-fx)
2. Generate images using the prompts in IMAGES.md
3. Download the generated images
4. Rename them to match the filenames above
5. Upload them to this /images folder in GitHub

### Option 2: Use Existing Photos
1. Find or take photos that match the scenes
2. Resize them to ~800x600px
3. Rename them to match the filenames above
4. Upload them to this folder

### Option 3: Temporary Placeholders
Until you add real images, the game will show a fallback placeholder image. The game is fully functional even without custom images.

## Uploading to GitHub

1. Go to your GitHub repository
2. Navigate to the `/images` folder
3. Click "Add file" > "Upload files"
4. Drag and drop your 5 images
5. Commit the changes

The game will automatically use the new images once they're uploaded!
